# IPTV 订阅源

自动从 iptvs-app 获取并测速的 IPTV 直播源。

## 订阅地址

在 TVBox / 影视仓 / 任意 IPTV 播放器中添加：

**M3U8 格式：**
```
https://raw.githubusercontent.com/mnyuer/iptv-sub/main/iptv.m3u8
```

**TXT 格式：**
```
https://raw.githubusercontent.com/mnyuer/iptv-sub/main/iptv.txt
```

## 更新频率

由 iptvs-app 自动测速更新，每 6 小时刷新一次，本仓库通过 crontab 自动同步。
